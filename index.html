⚡ Overlay creator temps réel avec Firebase

Crée ton overlay de stream, connecte-le à OBS et pilote-le avec un simple lien.

Crée un compte avec un identifiant + mot de passe hashé côté client, édite ton overlay, puis récupère une URL du style ?room=ton-id pour l'utiliser dans OBS, Streamlabs ou un navigateur en source web.

Ce que ça fait

Compte ID + mot de passe
Lien OBS ?room=ton-id
Sync live Realtime DB

Connexion / création de compte

Le mot de passe est hashé en SHA-256 dans la base. Comme l'app est 100% front, ce hashage est une protection basique côté client, pas un remplacement d'un vrai backend sécurisé.

Astuce: ton identifiant devient aussi la room publique lue par l'overlay.
